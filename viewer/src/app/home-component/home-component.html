This website displays the Eusebian canons of the New Testament using the text of the SBL Greek New Testament.

The Greek passages are aligned visually for easy comparison. When you view by canon, you see all of the texts in that canon table. When you view by gospel, you see the entire text of the gospel, along with corresponding passages where they exist. (For Canon 10—which has passages that only occur in one gospel—the alignment of the texts is not meaningful; it was just convenient to display the texts in the same format.)

All of the data (in JSON and SQLite format), along with the code for this website, are available in the GitHub repository.
